## Installation

**Install or open the Vs Code.**

*  **Run CLI**

```CLI-step 1
npm i
```

```CLI-step 2
mysql -u root -p
```

```CLI-step 3 
source ./db/schema.sql
```

```CLI-step 4
 npm run seed 
 ```

 ```CLI-step 5
 npm start
 ```   

```CLI-step 6
node server.js
```
